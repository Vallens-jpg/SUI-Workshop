# BLIXA Landing Page

Platform cetak aman on-chain pertama di dunia yang menggunakan teknologi blockchain Sui.

## Fitur

- **Desain Modern**: Terinspirasi dari Cetus.zone dengan gradient teal/cyan yang elegan
- **Responsive**: Tampilan optimal di semua perangkat
- **Animasi Smooth**: Fade-in animations dan parallax effects
- **Informasi Lengkap**: Mencakup semua aspek BLIXA dari masalah hingga solusi

## Struktur File

```
blixa-landing/
├── index.html      # Struktur HTML utama
├── styles.css      # Styling dengan tema dark teal
├── script.js       # Interaktivitas dan animasi
└── README.md       # Dokumentasi ini
```

## Cara Menggunakan

1. Buka file `index.html` di browser
2. Atau gunakan live server untuk development

```bash
# Jika menggunakan Python
python -m http.server 8000

# Jika menggunakan Node.js
npx serve
```

3. Akses di browser: `http://localhost:8000`

## Customization

### Warna
Edit variabel CSS di `styles.css`:
```css
:root {
    --primary-color: #00D9C0;
    --secondary-color: #0A4A4A;
    --dark-bg: #0A1F1F;
    --darker-bg: #051414;
}
```

### Konten
Edit teks langsung di `index.html` sesuai kebutuhan.
